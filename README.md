### 💡 Troubleshooting

- **Installation fails:** Run as Administrator/sudo and temporarily disable any antivirus software.
- **Simulator won't launch:** Ensure you have the latest version of your operating system.
- **Performance issues:** Close other background applications, especially browser tabs.
- **MOMOs not appearing:** Use the "Refresh MOMO Database" button within the tool's settings menu.
